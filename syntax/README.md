# Go's syntax
## 1. Declare Variables
### a. Using types
```go
var name string
var age int
var gender string
name = "Teddy"
age = 21
gender = "Male"
```

### b. Using typeinference
  1. Using var
  ```go
  var name = "Teddy"
  var age = 21
  var gender = "Male"
  ```

  2. Using :=
  ```go
  name := "Teddy"
  age := 21
  gender := "Male"
  ```

*:= can't be used outside functions*
